# داشبورد پروژه‌های فعال سازمان (Dash & Plotly)

این نمونه، یک داشبورد سه‌سطحی برای نمایش وضعیت پروژه‌های فعال یک سازمان است که با استفاده از **Dash**، **Plotly** و **pandas** توسعه داده شده و شامل داده‌های نمونه‌ی واقع‌گرایانه‌ی فارسی (واحدها، مدیران و پروژه‌ها) می‌باشد.

## اجرای پروژه

1. در ترمینال به پوشه‌ی پروژه بروید:

```bash
cd e:\projects\dashboard
```

2. (اختیاری ولی پیشنهاد‌شده) یک محیط مجازی بسازید و فعال کنید.

3. وابستگی‌ها را نصب کنید:

```bash
pip install -r requirements.txt
```

4. برنامه را اجرا کنید:

```bash
python app.py
```

5. مرورگر را باز کرده و آدرس زیر را باز کنید:

```text
http://127.0.0.1:8050/
```

## سطوح داشبورد

- **سطح ۱ (نمای اصلی)**:
  - نمودار میله‌ای تعداد پروژه‌های فعال هر واحد (`projects-per-dept-bar`)
  - نمودار خطی میانگین امتیاز پروژه‌ها برای هر واحد (`avg-score-line`)
- **سطح ۲ (پس از کلیک روی نوار یک واحد)**:
  - جدول مدیران آن واحد با تعداد پروژه و میانگین امتیاز (`dept-managers-table`)
- **سطح ۳ (پس از کلیک روی نام مدیر در جدول سطح ۲)**:
  - جدول جزئیات پروژه‌های همان مدیر شامل نام پروژه، تاریخ شروع، وضعیت و امتیاز (`manager-projects-table`)

رابط کاربری با Bootstrap و با جهت‌دهی راست‌به‌چپ طراحی شده تا برای داده‌های فارسی ظاهری حرفه‌ای و قابل‌خواندن داشته باشد.
